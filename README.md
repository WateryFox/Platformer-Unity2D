I developed this project to explore the fundamentals of 2D game design and character mechanics within the Unity engine. The goal was to create a classic platforming experience where the player must navigate through a level, collecting coins and finding hidden chests to successfully reach the finish line. I wanted to focus on how to make a character feel responsive and how to use triggers and collisions to create interactive world elements. This project served as my primary introduction to C# scripting and the Unity component system.

The core of the gameplay is built around a custom player controller that handles horizontal movement and jumping logic. I used Unity’s Rigidbody2D and BoxCollider2D components to manage the physics, ensuring that the character interacts correctly with the ground and obstacles. One of the most important parts of the development was setting up the "Layer Mask" system so the player can only jump when touching the ground, preventing infinite air jumps. I also implemented a collection system where coins and chests act as triggers; when the player overlaps with them, the script updates the game state and removes the item from the scene.

I wrote several C# scripts to handle the different parts of the game, including a Game Manager to track the score and a Level Loader for the transition to the finish screen. I spent a lot of time learning how to use the Unity Animator to sync the character's movement with walking and jumping animations. Handling the logic for the "Finish Line" was also a key part of the project, as it required checking if the player had met specific requirements before allowing the level to end. This taught me how to communicate between different scripts using tags and public variables.

Building this game gave me a much better understanding of the "Game Object" workflow and how to organize a complex project with multiple assets and scripts. If I were to expand on this, I would love to add enemy AI with basic patrol patterns and a more advanced UI that shows a real time coin counter on the screen. I am also interested in exploring "Tilemaps" to create more detailed and efficient level designs. This project was a major step in my journey toward game development, proving that I can take a concept from a blank scene to a fully playable experience.
